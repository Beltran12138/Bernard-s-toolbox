## [2.0.0] - 2025-01-20

### 🎉 重大升级

#### ✨ 新增功能

**架构层面:**
- 🗂️ **数据与视图分离** - 工具数据提取为独立的 `data/tools.json` 文件
- 📦 **模块化设计** - 数据/逻辑/样式清晰分离,便于维护
- 🔄 **动态加载** - 支持异步加载数据,错误处理完善

**核心功能:**
- ⌨️ **Command Palette** - VS Code 风格的命令面板 (Ctrl+K)
  - 快速搜索工具
  - 执行系统命令 (导出/重置/统计)
  - 支持键盘导航
- 🔍 **Fuse.js 模糊搜索** - 智能语义匹配,支持拼写容错
- 🎯 **Workflow 场景预设** - 4 个预置工作流快速切换
  - DEEP_WORK (专注编程)
  - CRYPTO_RESEARCH (加密研究)
  - JOB_HUNTING (求职准备)
  - CONTENT_CREATION (内容创作)
- 📊 **实时状态监控**
  - GitHub API 在线状态检测 (30秒间隔)
  - BTC 实时价格显示 (1分钟更新)
  - 工具总数统计
- 🎭 **Boss Key** - Shift+Esc 快速切换到 Google 页面

**视觉优化:**
- 🎨 **噪音纹理背景** - SVG Noise 增加模拟信号质感
- 🧲 **卡片磁吸效果** - 悬停时 3D 变换 + 阴影增强
- ✨ **边框流光动画** - 渐变边框呼吸效果
- ⭐ **评分系统** - 5星推荐度 + 难度等级 (⚡符号)
- 🏷️ **多维标签显示** - 每个工具显示 2 个关键标签
- 🎞️ **优化动画** - 卡片淡入动画时序优化

**交互增强:**
- 🔔 **Toast 通知系统** - 操作反馈提示
- 🎹 **快捷键支持**
  - Ctrl+K: 命令面板
  - Esc: 关闭面板
  - Shift+Esc: Boss Key
- 🖱️ **点击外部关闭** - Command Palette 支持点击背景关闭

#### 📊 数据增强

**工具数据扩展:**
- 新增 `id` 字段 (唯一标识符)
- 新增 `tags` 数组 (多维标签)
- 新增 `rating` 字段 (推荐度 1-5)
- 新增 `difficulty` 字段 (难度等级 1-5)

**Workflow 系统:**
- 定义 4 个预置工作流
- 支持自定义图标和颜色
- 工具关联通过 ID 引用

**分类优化:**
- 统一分类命名 (去除 "待探索" 等模糊分类)
- 新增明确分类:
  - 开发工具
  - 升学求职
  - 实习训练营
  - 娱乐
  - 等...

#### 🎨 视觉细节

**字体优化:**
- JetBrains Mono 等宽字体加载优化
- FontAwesome 6.4.0 图标库

**颜色系统:**
- 极客绿 (`#00dc82`) 作为主色调
- 深色背景 (`#050505`) 极致对比
- 精心调配的次要色系

**响应式改进:**
- 移动端优化布局
- 触摸友好的交互元素
- 自适应卡片网格

#### 🛠️ 技术改进

**性能优化:**
- 模糊搜索引擎 (Fuse.js)
- 事件监听去抖优化
- 动画性能提升

**错误处理:**
- 数据加载失败降级显示
- API 请求超时处理
- JSON 格式验证

**代码质量:**
- 函数模块化
- 注释完善
- 变量命名规范

#### 📚 文档完善

新增文档:
- ✅ `README.md` - 完整使用文档
- ✅ `QUICKSTART.md` - 5分钟快速上手
